# Iceberg Demo Package

Local stand-in for Intact's Iceberg design system.

In the live environment, use an opensrc-style source fetch for the real package before assuming component APIs. This demo package only provides tokens and CSS utility classes so the Angular app can build without private registry access.

## Exports

- `@intact/iceberg/tokens.css`
- `@intact/iceberg/styles.css`

## Useful Classes

- `.ib-shell`
- `.ib-panel`
- `.ib-field`
- `.ib-input`
- `.ib-select`
- `.ib-button`
- `.ib-button--primary`
- `.ib-error-list`
- `.ib-status-pill`
